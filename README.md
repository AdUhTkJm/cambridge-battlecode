# 目前进度

## 开局阶段

基地将会创造 6 个机器人。

### 寻路

每个机器人独立寻路。

最开始机器人处于 `EXPLORER` 状态，优先走向视野里的矿点，如果没有的话就随机，碰到障碍时反弹。

遇到矿点时，如果没有足够的 Ti，它将会切换到 `WAIT_FOR_HARVEST` 状态。当 Ti 足够多时，它会在矿点上建一个 harvester，然后切换到 `BUILD_CONVEYOR` 状态，把传送带拉到基地。

在拉取传送带时，它将会直接计算出整个路径，并沿着这条路径走一遍。但是有些问题：

- 拉传送带的时候是 4 个方向的，而平常走是 8 个方向的，这意味着传送带的路径和走过来的路径可能大相径庭（而且可能涉及到机器人所不知道的地图）；

- 无法重复利用传送带，而且每个矿点只有一个传送带；

- 如果寻路失败，会一直卡住。
